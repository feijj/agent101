🧠 Claude Code 之路

国产大模型原生支持 Anthropic API，直接对接 Claude Code。

工作原理

Claude Code 是 Anthropic 公司的 AI 编程助手。所有国产大模型平台都已经原生支持 Anthropic API 格式，只需要在配置文件中指向对应平台的 /anthropic 端点即可，无需任何中间代理。 

Claude Code  ──→  国产大模型平台的 /anthropic 端点
             直接连接，无需任何代理
🍎 Mac 用户必读 苹果电脑可能会阻止你运行脚本。解决方法：按住 Control 键 → 点击脚本文件 → 选择「打开」→ 再点「打开」。首次运行后即可正常双击。

1 准备运行环境

脚本会检查 Node.js 并配置国内加速镜像。

双击运行：

Windows: tools/windows/1-setup-env.bat

Mac: tools/mac/1-setup-env.command

2 安装并配置

安装 Claude Code 并选择你的 AI 大脑。脚本会自动将配置写入 ~/.claude/settings.json。

💡 如何获取 API Key 点击上面的链接 → 注册账号 → 找到「API 密钥」或「API Keys」页面 → 点击「创建」→ 复制生成的密钥。

复制好 API Key 后，双击运行：

Windows: tools/windows/2-install-and-configure.bat

Mac: tools/mac/2-install-and-configure.command

3 启动！

选择工作目录并启动 Claude Code。

双击运行：

Windows: tools/windows/3-start.bat

Mac: tools/mac/3-start.command

🎮 试试你的第一个 Prompt 启动后，在 > 提示符后输入：

"帮我写一个可以在浏览器里玩的贪吃蛇游戏，界面要现代好看，用深色主题。"
